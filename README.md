# AI Study Assistant 🚀

An AI-powered study assistant built using React and Flask that helps students learn effectively using Google Gemini AI. The system guides users step-by-step instead of directly giving answers.

---

## ✨ Features

* 💬 Interactive AI Chatbot
* 🧠 Guided problem-solving approach
* 🎓 Adjustable difficulty levels (School to College)
* 📐 LaTeX support for mathematical expressions
* 📝 Markdown formatted responses
* ⚡ Real-time response generation
* 💭 Conversation memory
* 🎨 Clean and responsive UI

---

## 🚀 Tech Stack

**Frontend:**

* React.js

**Backend:**

* Python Flask

## ⚙️ Setup Instructions

### 1. Clone the repository

git clone https://github.com/Priya-singh03/ai-study-assistant.git

### 2. Backend Setup

cd backend
pip install -r requirements.txt
python app.py

### 3. Frontend Setup

cd frontend
npm install
npm start

---

## 🔑 API Setup

Create a `.env` file in backend:

GEMINI_API_KEY=your_api_key
